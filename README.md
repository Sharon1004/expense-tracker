# 💸 Smart Expenses App

A clean and intuitive **expense tracking web application** built with React. Track your spending, manage budgets, and stay on top of your finances — all in one place.

🌐 **Live Demo:** [https://smart-expenses-app.netlify.app/](https://smart-expenses-app.netlify.app/)

---

## 📸 Preview

> Visit the live app to explore the full experience → [smart-expenses-app.netlify.app](https://smart-expenses-app.netlify.app/)

---

## ✨ Features

- ➕ Add, edit, and delete expenses
- 📂 Categorize expenses (Food, Travel, Shopping, etc.)
- 📊 Visual summary of your spending
- 🗓️ Filter expenses by date or category
- 💾 Persistent data storage (localStorage)
- 📱 Fully responsive — works on mobile & desktop

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or above)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/smart-expenses-app.git

# Navigate into the project directory
cd smart-expenses-app

# Install dependencies
npm install

# Start the development server
npm start
```

The app will run at `http://localhost:3000`

---

## 📦 Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build/` folder.

---

## 🌍 Deployment

This app is deployed via **Netlify**. To deploy your own version:

1. Push your code to GitHub
2. Go to [netlify.com](https://www.netlify.com/) and connect your repo
3. Set build command: `npm run build`
4. Set publish directory: `build`
5. Click **Deploy** 🎉
